The Persistence of Memory

Artist: Salvador Dalí

Year of Creation: 1931

Description:
The Persistence of Memory is one of Salvador Dalí's most famous surrealist paintings. It features melting clocks draped over objects in a dreamlike landscape, symbolizing the fluidity and relativity of time.
